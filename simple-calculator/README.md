# Simple Calculator

A simple command-line calculator written in C that handles basic arithmetic operations. This project is part of the low-level programming curriculum at Holberton School.

## Description

The program displays a menu allowing the user to choose an arithmetic operation (Addition, Subtraction, Multiplication, or Division) or quit the program. It safely handles division by zero and processes decimal numbers consistently using `double` precision variables.

## Requirements

* All files are compiled on Ubuntu 20.04 LTS using `gcc`.
* Compilation options: `-Wall -Werror -Wextra -pedantic -std=gnu89`
* Code adheres to the Betty style guide.

## File Structure

* **`calculator.c`**: The main source file containing the menu loop and arithmetic logic.
